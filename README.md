# Airsoft Rock Galactic

Una tienda ficticia dedicada a la venta de armas de **airsoft**, diseñada para informar, guiar y ayudar a los amantes de este deporte a encontrar su arma perfecta.  
El proyecto busca ofrecer una experiencia **moderna, intuitiva y centrada en el consumidor**.

---

## 🚀 Tecnologías utilizadas
- **HTML**
- **CSS**
- **JavaScript**
- **Tailwind CSS**

---

## 📂 Instalación y uso
No requiere instalación especial.  
Para visualizar el proyecto solo debes abrir el archivo `index.html` en cualquier navegador.

```bash
# Clonar el repositorio
git clone https://github.com/TU_USUARIO/airsoft-rock-galactic.git

# Ingresar al proyecto
cd airsoft-rock-galactic

# Abrir el index en un navegador
